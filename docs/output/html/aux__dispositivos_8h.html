int colocar_fila_sp ( fila_sp * f,
int valor )

Places a value into the serial port queue.

Inserts a value into the queue at the position indicated by the end index (fim). Increments the end index and the size of the queue. If the queue is full (i.e., the size of the queue is equal to its capacity), the function returns 1 to indicate an error.

Parameters
fPointer to the queue. This is the queue where the value will be placed.
valorThe value to place. This is the data that will be placed into the queue.
Returns
Returns 0 if the value was successfully placed into the queue, 1 if the queue is full.
