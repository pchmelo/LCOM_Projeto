int inserir_fila_sp ( fila_sp * f,
int rbr,
uint8_t status )

Inserts a value into the serial port queue.

Checks if the data is available by using the status parameter. If the data is available, it calls the colocar_fila_sp function to insert the value into the queue.

Parameters
fPointer to the queue. This is the queue where the value will be inserted.
rbrThe value to insert. This is the data that will be inserted into the queue.
statusThe status of the serial port. This is used to check if the data is available for insertion.
Returns
Returns 0 if the value was successfully inserted into the queue, 1 otherwise.
