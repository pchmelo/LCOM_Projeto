◆ botPlayCard()

int botPlayCard ( Card * playerDeck)

Determines the card to be played by the bot.

Decides which card the bot should play based on the current round suit and trump suit. If the round suit hasn't been set, it sets the round suit to the suit of the first card in the bot's deck and plays that card. If the bot has a card of the round suit, it plays that card. If the bot doesn't have a card of the round suit, but has a card of the trump suit, it plays that card. If the bot doesn't have a card of either the round suit or the trump suit, it plays the first card in its deck.

Parameters
playerDeckPointer to the array of cards in the bot's deck.
Returns
The index of the card to be played in the bot's deck.

Fills a deck of cards.

Initializes a deck of cards, assigning each card a rank, suit, value, and sprite. Fills the deck in a specific order, with the rank and suit incrementing sequentially. The value of the cards is determined by a switch statement, with specific values assigned to ranks 5-9. All other ranks are assigned a value of 0.

Sprite * getSprite ( int rank,
int suit )

Retrieves the sprite for a card based on its rank and suit.

Maps a card's rank and suit to it's sprite. The rank is mapped to a number from 0 to 9, representing 2, 3, 4, 5, 6, Q, J, K, 7, and Ace respectively. The suit is mapped to a number from 0 to 3, representing clubs, diamonds, hearts, and spades respectively. If the rank or suit does not match any of the predefined values, the function returns NULL.

Parameters
rankThe rank of the card.
suitThe suit of the card.
Returns
A pointer to the sprite representing the card, or NULL if the rank or suit is invalid.
