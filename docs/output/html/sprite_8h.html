typedef struct Sprite Sprite

Structure representing a sprite.

Contains the x and y coordinates of the sprite, its width and height, and a pointer to its pixel data.

void alocar_sprite_mem ( )

Allocates memory for sprites.

Calls the criar_sprite function for each sprite in the game or application. Each sprite is loaded from an XPM file and stored in memory.

◆ criar_sprite()

Sprite * criar_sprite ( xpm_map_t ficheiro_xpm)

Creates a sprite from an XPM file.

Creates a sprite by loading an XPM file into memory. Allocates memory for the sprite, loads the XPM file, and sets the sprite's width and height. If there is an error during memory allocation or loading the XPM file, the function prints an error message and returns NULL.

Parameters
ficheiro_xpmThe XPM file to load.
Returns
A pointer to the created sprite, or NULL if there was an error.

◆ destruir_sprite()

void destruir_sprite ( Sprite * sprite)

Destroys a sprite.

Frees the memory allocated for a sprite and its pixel data. If the sprite is NULL, the function prints an error message and returns. After freeing the memory, it sets the sprite pointer to NULL.

Parameters
spriteThe sprite to destroy.
