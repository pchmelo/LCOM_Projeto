Processes keyboard inputs based on the current game state.

Handles the processing of keyboard inputs based on the current game state. If the game is in the main menu, processes the keyboard inputs for the main menu, connection, or invitation based on the current connection and invitation states. If the game is in the game state or multiplayer mode, it processes the keyboard inputs for the game.
