◆ process_round_MUL()

void process_round_MUL ( )

Processes the round in a multiplayer game.

Handles the processing of a round in a multiplayer game. Checks if the turn is 5, indicating the end of a round, and then processes the round to determine the winner. Also handles the resetting of the game state at the end of a round. If the turn is not 5, processes the round for each player, waiting for 60 seconds before moving to the next player.
