int criar\+\_\+jogo (\begin{DoxyParamCaption}{}{}\end{DoxyParamCaption})}



Manages the game state. 

Sets up the game by clearing the graphic buffer and printing the background. Depending on the game state, it manages player hands, displays the score board and the trump card. Handles card selection, card grabbing, and card display based on the round and grab states.

\begin{DoxyReturn}{Returns}
0 on success, 1 on error.
